반응형 웹을 손쉽게 만들기 위해 그리드 시스템을 사용함

각 장치에 적합하게 웹 페이지를 구현하기 위해 장치별 특정 해상도를 기준으로 레이아웃을 다르게 표현하는 것을 의미함

indicius.com

레이아웃 클래스 혼합 설정

하나의 요소애 레이아웃 클래스를 여러개를 설정할 수도 있음

d-*-none:특정 화면 크기일때 요소를 화면에서 보이지 않게 함

d-*-block:특정 화면 크기일때 요소를 화면에서 보이게 함

d-block:특정 요소를 화면에서 무조건 보이게 함

d-none:특정 요소를 화면에서 무조건 보이지 않게 함
